JVM内存模型
· 2019-07-06 · # JVM

Java RunTime Data Area(Java 运行时数据区) 分为:

  • Java Heap(Java堆)
  • Method Area(方法区) (Java 8 之前)
  • Meta Space(元空间) (Java 8 之后)
  • VM Stack(虚拟机栈)
  • Native Methiod Stack(本地方法栈)
  • Program Counter Register(程序计数器)。
